## Gradient and Shadow

```dart
ShadButton(
  onPressed: () {},
  gradient: const LinearGradient(colors: [
    Colors.cyan,
    Colors.indigo,
  ]),
  shadows: [
    BoxShadow(
      color: Colors.blue.withOpacity(.4),
      spreadRadius: 4,
      blurRadius: 10,
      offset: const Offset(0, 2),
    ),
  ],
  child: const Text('Gradient with Shadow'),
)
```

</Preview>

## Troubleshoot

### iconSize

The size of the icon, defaults to `Size.square(16)`.

:::caution
The icon size is applied only if you use a `ShadImage` as `icon`. If you use something else, wrap it with a `SizedBox`.
:::
